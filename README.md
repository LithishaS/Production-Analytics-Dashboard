# 📊 Production Analytics Dashboard

An interactive production analytics dashboard built using HTML, CSS, JavaScript, and Chart.js.

## 📌 Project Overview

This dashboard was created to analyze production, defect, supplier, and operator performance data. It allows users to upload CSV or Excel files and view key business metrics through interactive charts.

## ✨ Features

- CSV/Excel file upload
- KPI cards
- Production quantity analysis
- Scrap percentage analysis
- Supplier contribution chart
- On-time rate tracking
- Pareto analysis for top defects
- Operator performance dashboard
- SPC-style cycle time analysis
- Interactive Chart.js visualizations

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
